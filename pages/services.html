👨‍💻 For Developers

Build on Yearn's
Infrastructure

Yearn's open-source codebase and extensive documentation make it easy to integrate with or build on top of the Yearn ecosystem.

📄

ERC-4626 Standard

All Yearn v3 vaults implement the ERC-4626 tokenized vault standard, ensuring seamless composability with any DeFi protocol that supports this interface. Build integrations once, deploy across the entire ERC-4626 ecosystem.

🔓

Open Source

All Yearn contracts are open source and available on GitHub. The codebase includes comprehensive NatSpec documentation, test suites, and deployment scripts to accelerate integration and strategy development.

🌐

Multi-Chain Deployment

Yearn v3 is deployed on Ethereum, Arbitrum, Optimism, Base, and Polygon. Developers can interact with Yearn vaults on any supported chain using the same interface and integration patterns.

💰

Revenue Sharing

Strategy developers earn performance fees on the yield generated by their strategies. Protocols that integrate Yearn vaults can negotiate fee sharing arrangements that benefit both parties and their users.
